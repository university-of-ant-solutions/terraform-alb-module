These are the resources which created:

* EC2-VPC Security Group

* ALB

## Usage example

Here's the gist of using it in local:

```
module "alb" {
  source = "./modules/alb"

  vpc_id            = "${var.vpc_id}"
  subnets           = "${var.subnets}"
  service_name      = "${var.service_name}"
}
```

1. Always `terraform plan` to see your change before running `terraform apply`.
